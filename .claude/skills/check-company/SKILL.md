# /check-company — Проверка контрагента

## Алгоритм
1. Вызови `search_company` по ИНН/ОГРН
2. Вызови `get_brief_report` для краткого отчёта
3. Вызови `get_analytics` для финансовой аналитики
4. Вызови `check_reliability` для оценки надёжности
5. Покажи сводку с оценкой риска

## Формат ответа
```
## Контрагент: [название]
**ИНН**: ... | **Статус**: действующая
**Риск**: НИЗКИЙ / СРЕДНИЙ / ВЫСОКИЙ
**Финансы**: выручка, прибыль
```

## Примеры
```
/check-company 7707083893
```
